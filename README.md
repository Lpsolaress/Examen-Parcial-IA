# Reconocimiento de Cartas con OpenCV

Este proyecto utiliza OpenCV para detectar cartas en un tapete verde, recortar la carta completa, y reconocer el número (rank) y el palo (suit) de la carta. El sistema utiliza plantillas predefinidas para realizar la detección y comparación.

---

## Requisitos

Antes de ejecutar el código, asegúrate de tener instalados los siguientes requisitos:

### Dependencias
- Python 3.6 o superior
- OpenCV
- NumPy
- Imutils

### Instalación de dependencias
Ejecuta el siguiente comando para instalar las dependencias necesarias:
```bash
pip install opencv-python opencv-python-headless numpy imutils

### Ejecución del programa
python3 Deteccion\ de\ Cartas.py --templates ./templates --camera 1
